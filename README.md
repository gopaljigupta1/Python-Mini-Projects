# 🎮 Python Mini Projects

## 📌 Overview

This repository contains a collection of beginner-friendly Python projects, including fun games and a simple café management system.

These projects are designed to strengthen core programming concepts such as **loops, conditionals, functions, and user interaction** while building practical and interactive applications.

---

## 🎯 Objective

The objective of this project is to practice and demonstrate Python fundamentals by building small real-world and game-based applications.

It aims to:

* Improve problem-solving skills
* Understand user input and program flow
* Build interactive command-line applications
* Apply logic using conditions and loops

---

## 🧩 Projects Included

### ☕ Café Management System

A simple system that simulates a café ordering process.

**Features:**

* Menu display
* Order selection
* Bill calculation
* Multiple item ordering

---

### 🎲 Dice Game

A random number-based game where players roll dice.

**Features:**

* Random dice generation
* Player vs computer logic
* Score comparison

---

### ✊✋✌️ Stone Paper Scissors Game

A classic game played between user and computer.

**Features:**

* User input vs computer choice
* Random selection using Python
* Win/Lose/Draw logic

---

## 🛠️ Technologies Used

* Python 🐍
* Random Module
* Basic CLI (Command Line Interface)
